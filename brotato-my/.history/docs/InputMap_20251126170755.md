# Godot 输入映射说明 (Input Map)

> 本文档记录了本项目使用的输入映射配置，兼容 Godot 4.x 版本。  
> 按钮命名遵循 Xbox 手柄标准，方便理解和维护。

## 移动控制 (Movement)

支持键盘 (WASD / 方向键) 和手柄 (左摇杆 / 十字键)。

| 动作名称       | 键盘按键           | 手柄输入                         | 说明     |
| :------------- | :----------------- | :------------------------------- | :------- |
| **move_up**    | `W`, `Up Arrow`    | 左摇杆上 (Axis 1 -), D-Pad Up    | 向上移动 |
| **move_down**  | `S`, `Down Arrow`  | 左摇杆下 (Axis 1 +), D-Pad Down  | 向下移动 |
| **move_left**  | `A`, `Left Arrow`  | 左摇杆左 (Axis 0 -), D-Pad Left  | 向左移动 |
| **move_right** | `D`, `Right Arrow` | 左摇杆右 (Axis 0 +), D-Pad Right | 向右移动 |

## UI 导航 (UI Navigation)

用于菜单界面的导航。

| 动作名称    | 键盘按键           | 手柄输入    | 说明     |
| :---------- | :----------------- | :---------- | :------- |
| **ui_up**   | `W`, `Up Arrow`    | D-Pad Up    | 菜单上移 |
| **ui_down** | `S`, `Down Arrow`  | D-Pad Down  | 菜单下移 |
| **ui_left** | `A`, `Left Arrow`  | D-Pad Left  | 菜单左移 |
| **ui_right**| `D`, `Right Arrow` | D-Pad Right | 菜单右移 |

## Xbox 按钮映射 (Button Actions)

按照 Xbox 手柄按钮命名，清晰直观。

| 动作名称       | 键盘按键           | 手柄按钮                    | 说明               |
| :------------- | :----------------- | :-------------------------- | :----------------- |
| **btn_a**      | `Enter`, `Space`   | A 键 (按钮 0)               | 确认/交互          |
| **btn_b**      | `Esc`              | B 键 (按钮 1)               | 取消/返回          |
| **btn_x**      | `E`                | X 键 (按钮 2)               | 特殊动作 1         |
| **btn_y**      | `F`                | Y 键 (按钮 3)               | 特殊动作 2/信息    |
| **btn_start**  | `Esc`              | Start (按钮 6)              | 暂停/菜单          |
| **btn_back**   | `Tab`              | Back/Select (按钮 10)       | 选择/视图切换      |
| **btn_lb**     | `Q`                | LB 左肩键 (按钮 4)          | 左肩功能           |
| **btn_rb**     | `R`                | RB 右肩键 (按钮 5)          | 右肩功能           |
| **btn_lt**     | `Shift`            | LT 左扳机 (按钮 9 & Axis 4) | 左扳机/冲刺        |
| **btn_rt**     | `Ctrl`             | RT 右扳机 (按钮 10 & Axis 5)| 右扳机/射击        |

## 右摇杆控制 (Right Stick)

用于视角控制、瞄准等。

| 动作名称               | 手柄输入          | 说明       |
| :--------------------- | :---------------- | :--------- |
| **stick_right_up**     | 右摇杆上 (Axis 3 -)| 右摇杆上   |
| **stick_right_down**   | 右摇杆下 (Axis 3 +)| 右摇杆下   |
| **stick_right_left**   | 右摇杆左 (Axis 2 -)| 右摇杆左   |
| **stick_right_right**  | 右摇杆右 (Axis 2 +)| 右摇杆右   |

## 如何在代码中使用

```gdscript
# 移动控制（支持手柄摇杆的模拟量输入）
func _physics_process(delta):
    var input_dir = Input.get_vector("move_left", "move_right", "move_up", "move_down")
    velocity = input_dir * SPEED
    move_and_slide()

# 按钮事件
func _input(event):
    if event.is_action_pressed("btn_a"):
        print("A 键：确认")
    
    if event.is_action_pressed("btn_b"):
        print("B 键：取消")
    
    if event.is_action_pressed("btn_x"):
        print("X 键：特殊动作")
    
    if event.is_action_pressed("btn_y"):
        print("Y 键：查看信息")
    
    if event.is_action_pressed("btn_start"):
        get_tree().paused = !get_tree().paused  # 暂停切换
    
    # 右摇杆控制（例如瞄准方向）
    var aim_dir = Input.get_vector(
        "stick_right_left", 
        "stick_right_right", 
        "stick_right_up", 
        "stick_right_down"
    )
    if aim_dir.length() > 0.1:
        # 使用右摇杆控制瞄准
        rotation = aim_dir.angle()
```

## 命名规范说明

- **`move_*`**: 角色移动相关（通常是左摇杆）
- **`ui_*`**: UI 界面导航
- **`btn_*`**: 手柄按钮动作（遵循 Xbox 命名）
- **`stick_right_*`**: 右摇杆控制（通常用于瞄准/视角）
