# Godot 输入映射说明 (Input Map)

> 本文档记录了本项目使用的输入映射配置，兼容 Godot 4.x 版本。

## 移动控制 (Gameplay)

支持键盘 (WASD / IJKL / 方向键) 和手柄 (左摇杆 / 十字键)。

| 动作名称 | 键盘按键 | 手柄输入 | 说明 |
| :--- | :--- | :--- | :--- |
| **move_up** | `W`, `I`, `Up Arrow` | 左摇杆上 (Axis 1 -), D-Pad Up | 向上移动 |
| **move_down** | `S`, `K`, `Down Arrow` | 左摇杆下 (Axis 1 +), D-Pad Down | 向下移动 |
| **move_left** | `A`, `J`, `Left Arrow` | 左摇杆左 (Axis 0 -), D-Pad Left | 向左移动 |
| **move_right** | `D`, `L`, `Right Arrow` | 左摇杆右 (Axis 0 +), D-Pad Right | 向右移动 |

## UI 导航 (UI Navigation)

用于菜单界面的导航。

| 动作名称 | 键盘按键 | 手柄输入 | 说明 |
| :--- | :--- | :--- | :--- |
| **ui_up** | `W`, `I`, `Up Arrow` | D-Pad Up | 菜单上移 |
| **ui_down** | `S`, `K`, `Down Arrow` | D-Pad Down | 菜单下移 |
| **ui_left** | `A`, `J`, `Left Arrow` | D-Pad Left | 菜单左移 |
| **ui_right** | `D`, `L`, `Right Arrow` | D-Pad Right | 菜单右移 |
| **ui_accept** | `Enter`, `Space`, `Kp Enter` | A 键 (Xbox) / X 键 (PS) | 确认/选择 |
| **ui_cancel** | `Esc` | B 键 (Xbox) / O 键 (PS) | 取消/返回 |
| **ui_select** | `E` | X 键 (Xbox) / Square 键 (PS) | 特殊选择 |
| **ui_focus_prev**| `Shift + Tab` | - | 焦点切换 |
| **ui_info** | `F` | Y 键 (Xbox) / Triangle 键 (PS) | 查看信息 |
| **ui_pause** | `Esc` | Start / Options | 暂停菜单 |

## 辅助功能 / 调试

| 动作名称 | 键盘按键 | 手柄输入 | 说明 |
| :--- | :--- | :--- | :--- |
| **open_bug_report** | `F10` | - | 打开 Bug 报告界面 |

## 特殊手柄控制

| 动作名称 | 手柄输入 | 说明 |
| :--- | :--- | :--- |
| **rjoy_up** | 右摇杆上 (Axis 3 -) | 右摇杆控制 |
| **rjoy_down** | 右摇杆下 (Axis 3 +) | 右摇杆控制 |
| **rjoy_left** | 右摇杆左 (Axis 2 -) | 右摇杆控制 |
| **rjoy_right** | 右摇杆右 (Axis 2 +) | 右摇杆控制 |
| **ltrigger** | LT / L2 (Button 4 & Axis 4) | 左扳机键 |
| **rtrigger** | RT / R2 (Button 5 & Axis 5) | 右扳机键 |

## 如何在代码中使用

```gdscript
func _physics_process(delta):
    # 获取移动向量 (支持手柄摇杆的模拟量)
    var input_dir = Input.get_vector("move_left", "move_right", "move_up", "move_down")
    velocity = input_dir * SPEED
    move_and_slide()

func _input(event):
    if event.is_action_pressed("ui_accept"):
        print("确认键被按下")
```
