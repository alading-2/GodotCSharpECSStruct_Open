## Node 扩展方法

如果使用 Godot 4.2+，可以通过扩展类（Extension Classes）为 Node 添加便捷方法：

```
# 扩展 Node 类
@tool
extends Node

# 扩展方法：安全设置对象池引用
func set_object_pool(pool: ObjectPool) -> 
void:
    set_meta("_object_pool", pool)

# 扩展方法：安全获取对象池引用
func get_object_pool() -> ObjectPool:
    var pool = get_meta("_object_pool")
    return pool if pool is ObjectPool and 
    is_instance_valid(pool) else null

# 扩展方法：直接回收自己到对象池
func return_to_pool() -> void:
    var pool = get_object_pool()
    if pool:
        pool.return_instance(self)
```
