# Language Preference
You MUST output all content in Simplified Chinese (简体中文).
This applies to:
- Implementation Plans (实施计划)
- Task Lists (任务列表)
- Code comments (代码注释)
- Chat responses (对话回复)
- Commit messages (提交信息)

Even if the technical terms are in English, the surrounding explanation and structure must be in Chinese.
