## 🚀 快速入门

### 打开插件

1. 在 Godot 编辑器顶部菜单栏点击 **「资源表」** 标签
2. 在路径输入框中输入资源文件夹路径，例如：
   ```
   res://Data/Data/Ability/Resource/
   ```
3. 按回车键加载，表格将自动列出该文件夹下所有 `.tres` 资源文件

### 基础编辑流程

| 步骤 | 操作 | 说明 |
|:---|:---|:---|
| 1 | **选中单元格** | 单击单元格，按 `Shift+Click` 多选，按 `Ctrl+Click` 单选/取消 |
| 2 | **编辑数值** | 底部面板显示对应的编辑器（数字/颜色/数组等） |
| 3 | **输入数值** | 方法一：底部面板**值区域单击**直接输入（回车确认）<br>方法二：右侧检查器修改 |
| 4 | **批量修改** | 选中整列后，底部面板修改会应用到所有选中行 |

## 🖥️ 界面详解

### 顶部工具栏

```
┌────────────────────────────────────────────────────────────────┐
│  资源路径: [res://Data/.../      ] [▼历史路径] [文件夹按钮]    │
│  [设置] [关于] [↑导入/导出 CSV]                                  │
│  过滤器: [GDScript Filter...] [类 ▼] [Sub: ☑] [Process: ▼]    │
├────────────────────────────────────────────────────────────────┤
│  [复选框▼] [Page ▼] [←] [→] [End]        每页: [50 ▼] 行      │
└────────────────────────────────────────────────────────────────┘
```

| 控件 | 功能 |
|:---|:---|
| **资源路径** | 输入要浏览的资源文件夹路径 |
| **历史路径下拉** | 快速切换最近打开的路径 |
| **设置按钮** | 列显示设置（显示/隐藏列） |
| **CSV 导入/导出** | 批量导入/导出数据到 CSV 文件 |
| **过滤器** | 按类名筛选资源类型 |
| **分页控件** | 资源较多时分页显示 |

### 表格区域

```
┌──────────┬──────────┬──────────┬──────────┐
│ 资源名称  │ 技能等级  │ 冷却时间  │ 施法距离  │  ← 列头（中文显示）
├──────────┼──────────┼──────────┼──────────┤
│ Fireball │    1     │   5.0    │   300    │
│ IceNova  │    1     │   8.0    │   250    │  ← 行 = 一个资源文件
│ Dash     │    1     │   4.0    │   200    │
└──────────┴──────────┴──────────┴──────────┘
```

| 操作 | 效果 |
|:---|:---|
| 单击单元格 | 选中该单元格，右侧检查器跳转对应资源 |
| 单击列头 | 按该列排序 |
| 右键列头 | 打开菜单（选择整列 / 隐藏列 / 打开子资源） |
| `Shift+Click` | 区间多选 |
| `Ctrl+Click` | 单选/取消选择 |

### 底部编辑面板（EDIT: Number）

当选中数字单元格时显示：

```
┌────────────────────────────────────────────────────────────────┐
│ EDIT: Number                    [AbilityCooldown]              │
├────────────────────────────────────────────────────────────────┤
│ ┌──────────────┐  ┌─────┬─────┬──────┬───────┬─────┬─────┐  │
│ │              │  │ +0.1│ +1  │ +10  │ +100  │ +X  │ *X  │  │
│ │    5.0       │  ├─────┼─────┼──────┼───────┼─────┼─────┤  │
│ │  (单击输入)   │  │ -0.1│ -1  │ -10  │ -100  │ -X  │ /X  │  │
│ └──────────────┘  └─────┴─────┴──────┴───────┴─────┴─────┘  │
│                      ↑ 快捷按钮    ↑ 自定义值 X = [1000]      │
│                                                                │
│  Fill with Sequence (?):  [Start*] [End] [Step] [Set] [Add]   │
└────────────────────────────────────────────────────────────────┘
```

#### 数字编辑区操作

| 操作 | 说明 |
|:---|:---|
| **单击数值区域** | 进入直接输入模式（类似检查器），输入值后回车确认 |
| **鼠标拖拽** | 左右拖动快速增减数值 |
| **快捷按钮** | `±0.1`、`±1`、`±10`、`±100` 快速调整 |
| **自定义值** | 使用 X 值进行 `+X` / `-X` / `*X` / `/X` 操作 |
| **序列填充** | 设置 Start/End/Step，批量生成等差数列填充选中单元格 |

## ✏️ 编辑操作

### 1. 单元格编辑

| 快捷键 | 功能 |
|:---|:---|
| `Tab` | 向右移动到下一个单元格 |
| `Shift+Tab` | 向左移动 |
| `↑` / `↓` | 上下移动（`Ctrl` 加速 ×10） |
| `Ctrl+C` | 复制选中单元格内容 |
| `Ctrl+V` | 粘贴到选中单元格 |
| `Delete` / `Backspace` | 清空内容 |
| `Ctrl+Z` / `Ctrl+Y` | 撤销 / 重做 |

### 2. 批量编辑

| 场景 | 操作 |
|:---|:---|
| 修改整列 | 点击列头选中整列 → 底部面板修改值 |
| 修改多行 | `Shift+Click` 选中多行 → 底部面板修改值 |
| 复制一列到另一列 | 选中源列 `Ctrl+C` → 选中目标列 `Ctrl+V` |

### 3. 行操作（右键菜单）

选中单元格后右键：
- **Rename** - 重命名资源文件
- **Duplicate** - 复制资源文件
- **Delete** - 删除资源文件

## 🔧 高级功能

### 1. 子资源打开

当列的类型是 Resource（资源引用）时：
- 右键列头 → **「Open Sub-Resources of Column」**
- 或选中单元格 → 右键 → **「Open Sub-Resources in Selection」**

表格会切换到显示这些子资源的内容。

### 2. 过滤与搜索

| 过滤器类型 | 说明 |
|:---|:---|
| **GDScript Filter** | 用 GDScript 表达式过滤行（高级） |
| **Class 下拉** | 按资源类名过滤 |
| **Condition/Process** | 按布尔/枚举值过滤 |

### 3. CSV 导入/导出

| 功能 | 路径 |
|:---|:---|
| 导出 CSV | 设置按钮 → Export CSV |
| 导入 CSV | 设置按钮 → Import CSV（注意：会覆盖资源！） |

## 🇨🇳 配置中文标签

中文列头标签由 `@/e:/Godot/Games/MyGames/复刻土豆兄弟/brotato-my/addons/resources_spreadsheet_view/column_labels_zh.gd` 文件控制。

### 添加新字段的中文标签

```gdscript
# 在 LABELS 字典中添加一行：
"字段名小写": "显示的中文",

# 示例：
"newdamage": "新伤害值",
"cooldownreduction": "冷却缩减(%)",
```

### 支持通配

插件使用 `to_lower()` 匹配键名，所以 `AbilityCooldown`、`abilityCooldown`、`abilitycooldown` 都能匹配到同一中文标签。

## 🐛 故障排除

| 问题 | 解决方案 |
|:---|:---|
| **中文标签不显示** | 检查 `column_labels_zh.gd` 是否有对应字段的映射 |
| **浮点数不显示 `.0`** | 已修复，整数值的 float 会显示 `5.0` |
| **无法连续输入数字** | 使用**底部面板的数值区域单击**进入输入模式，回车确认 |
| **修改没保存** | 按 `Enter` 或切换单元格触发保存；检查文件权限 |
| **列头不显示中文** | 确保所有 gd 文件编译无错误（检查 Godot 输出面板） |

## 💡 使用技巧

1. **快速跳转**：使用 `Tab` 在单元格间快速跳转编辑
2. **整列操作**：点击列头选中整列，批量修改同类型数据
3. **拖拽改值**：底部数字面板的值区域可拖拽滑动调整
4. **双击全选**：进入直接输入模式后，文字自动全选，直接输入即可覆盖
