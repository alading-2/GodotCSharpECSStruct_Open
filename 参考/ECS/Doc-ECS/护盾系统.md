# 现代 ECS 护盾系统设计方案

## 🎯 设计理念

基于现代游戏引擎（Unity ECS、Unreal Component）的设计原则，结合项目的 Entity-Component-Schema 架构，设计一个**数据与逻辑分离**、**职责清晰**、**高性能**的护盾系统。

### 核心原则

1. **数据与逻辑分离** - Schema 定义数据结构，DataManager 存储数据，LogicComponent 处理业务逻辑
2. **单一职责** - 每个组件只负责一个明确的功能领域
3. **依赖注入** - LogicComponent 通过 ComponentManager 获取所需的 DataManager
4. **事件驱动** - 组件间通过事件系统进行通信
5. **性能优先** - 简化数据结构，优化计算逻辑

## 📊 架构设计

```
Entity
├── DataManager (Inte_Shield) (数据存储)
│   ├── Schema: Inte_Shield
│   └── 纯数据CRUD操作
├── ShieldComponent (业务逻辑)
│   ├── 依赖: DataManager (Inte_Shield)
│   ├── 护盾管理逻辑
│   └── 伤害处理逻辑
└── DamageSystemComponent (伤害系统集成)
    ├── 依赖: ShieldComponent
    └── 伤害流程处理
```

## 🗃️ DataManager - 数据存储层

### DataManager (Inte_Shield)

```typescript
/**
 * 护盾数据组件
 * 专门负责护盾数据的存储和基础CRUD操作
 * 符合现代ECS架构的数据与逻辑分离原则
 */
export class Inte_Shield extends DataManager<Inte_Shield> {
  protected static readonly TYPE: string = "Inte_Shield";

  constructor(owner: Entity, initialData?: Partial<Inte_Shield>) {
    super(owner, SCHEMA_TYPES.SHIELD_DATA, initialData);
  }

  static getType(): string {
    return this.TYPE;
  }

  // ==================== 基础数据访问 ====================

  /**
   * 获取所有护盾
   */
  getShields(): ShieldInstance[] {
    return this.get("shields") || [];
  }

  /**
   * 设置护盾列表
   */
  setShields(shields: ShieldInstance[]): void {
    this.set("shields", shields);
    this.updateTotalValue(shields);
    this.set("lastUpdateTime", os.time());
  }

  /**
   * 添加护盾
   */
  addShield(shield: ShieldInstance): boolean {
    const shields = this.getShields();
    const maxCapacity = this.get("maxCapacity");

    if (shields.length >= maxCapacity) {
      return false;
    }

    shields.push(shield);
    this.setShields(shields);
    return true;
  }

  /**
   * 移除护盾
   */
  removeShield(shieldId: string): boolean {
    const shields = this.getShields();
    const index = shields.findIndex((s) => s.id === shieldId);

    if (index === -1) {
      return false;
    }

    shields.splice(index, 1);
    this.setShields(shields);
    return true;
  }

  /**
   * 根据ID获取护盾
   */
  getShieldById(shieldId: string): ShieldInstance | null {
    return this.getShields().find((s) => s.id === shieldId) || null;
  }

  /**
   * 获取护盾总值
   */
  getTotalValue(): number {
    return this.get("totalValue");
  }

  /**
   * 获取最大容量
   */
  getMaxCapacity(): number {
    return this.get("maxCapacity");
  }

  /**
   * 设置最大容量
   */
  setMaxCapacity(capacity: number): void {
    this.set("maxCapacity", capacity);
  }

  // ==================== 内部工具方法 ====================

  /**
   * 更新护盾总值
   */
  private updateTotalValue(shields: ShieldInstance[]): void {
    const total = shields
      .filter((s) => s.isActive)
      .reduce((sum, shield) => sum + shield.value, 0);
    this.set("totalValue", total);
  }
}
```

## ⚙️ LogicComponent - 业务逻辑层

### ShieldComponent.ts

```typescript
/**
 * 护盾逻辑组件
 * 负责护盾系统的业务逻辑处理
 * 通过依赖注入获取DataManager (Inte_Shield)
 */
export class ShieldComponent extends Component {
  protected static readonly TYPE: string = "ShieldComponent";

  // 依赖的数据组件
  private shieldData: Inte_Shield;

  // 依赖声明
  static readonly requiredDataManagers = ["Inte_Shield"];

  constructor(owner: Entity) {
    super(owner);
  }

  static getType(): string {
    return this.TYPE;
  }

  /**
   * 组件初始化
   */
  initialize(): void {
    // 通过依赖注入获取数据组件
    this.shieldData = this.owner.getDataManager("Inte_Shield");

    if (!this.shieldData) {
      throw new Error("ShieldComponent requires DataManager (Inte_Shield)");
    }

    // 启动护盾更新定时器
    this.startShieldUpdateTimer();
  }

  // ==================== 护盾管理API ====================

  /**
   * 添加护盾
   */
  addShield(
    value: number,
    type: ShieldType = ShieldType.UNIVERSAL,
    duration: number = -1,
    priority: number = 0
  ): string {
    const shield: ShieldInstance = {
      id: this.generateShieldId(),
      value: value,
      type: type,
      priority: priority,
      remainingTime: duration,
      createTime: os.time(),
      isActive: true,
    };

    const success = this.shieldData.addShield(shield);

    if (success) {
      // 触发护盾添加事件
      this.owner.emit("shield:added", { shield });
      return shield.id;
    }

    return "";
  }

  /**
   * 移除护盾
   */
  removeShield(shieldId: string): boolean {
    const shield = this.shieldData.getShieldById(shieldId);
    const success = this.shieldData.removeShield(shieldId);

    if (success && shield) {
      // 触发护盾移除事件
      this.owner.emit("shield:removed", { shield });
    }

    return success;
  }

  /**
   * 处理伤害
   * @param damage 伤害值
   * @param damageType 伤害类型
   * @returns 剩余伤害
   */
  processDamage(damage: number, damageType: string = "universal"): number {
    if (damage <= 0) return 0;

    let remainingDamage = damage;
    const shields = this.getActiveShieldsByPriority();
    const damagedShields: ShieldInstance[] = [];

    // 按优先级消耗护盾
    for (const shield of shields) {
      if (remainingDamage <= 0) break;

      // 检查护盾类型是否能吸收此类型伤害
      if (!this.canAbsorbDamage(shield, damageType)) continue;

      const absorbed = Math.min(remainingDamage, shield.value);
      shield.value -= absorbed;
      remainingDamage -= absorbed;
      damagedShields.push(shield);

      // 护盾破碎
      if (shield.value <= 0) {
        shield.isActive = false;
        this.owner.emit("shield:broken", { shield, damage: absorbed });
      }
    }

    // 更新护盾数据
    if (damagedShields.length > 0) {
      this.shieldData.setShields(this.shieldData.getShields());
      this.owner.emit("shield:damaged", {
        totalDamage: damage,
        absorbed: damage - remainingDamage,
        remaining: remainingDamage,
      });
    }

    return remainingDamage;
  }

  /**
   * 清空所有护盾
   */
  clearAllShields(): void {
    const shields = this.shieldData.getShields();
    this.shieldData.setShields([]);

    this.owner.emit("shield:cleared", { clearedShields: shields });
  }

  // ==================== 查询API ====================

  /**
   * 获取激活的护盾（按优先级排序）
   */
  getActiveShieldsByPriority(): ShieldInstance[] {
    return this.shieldData
      .getShields()
      .filter((s) => s.isActive)
      .sort((a, b) => a.priority - b.priority);
  }

  /**
   * 获取指定类型的护盾
   */
  getShieldsByType(type: ShieldType): ShieldInstance[] {
    return this.shieldData
      .getShields()
      .filter((s) => s.isActive && s.type === type);
  }

  /**
   * 获取护盾统计信息
   */
  getShieldStats() {
    const shields = this.shieldData.getShields();
    const activeShields = shields.filter((s) => s.isActive);

    return {
      total: shields.length,
      active: activeShields.length,
      totalValue: this.shieldData.getTotalValue(),
      byType: this.groupShieldsByType(activeShields),
    };
  }

  // ==================== 内部方法 ====================

  /**
   * 检查护盾是否能吸收指定类型的伤害
   */
  private canAbsorbDamage(shield: ShieldInstance, damageType: string): boolean {
    if (shield.type === ShieldType.UNIVERSAL) return true;
    if (shield.type === ShieldType.PHYSICAL && damageType === "physical")
      return true;
    if (shield.type === ShieldType.MAGIC && damageType === "magic") return true;
    return false;
  }

  /**
   * 按类型分组护盾
   */
  private groupShieldsByType(
    shields: ShieldInstance[]
  ): Record<string, number> {
    return shields.reduce((acc, shield) => {
      acc[shield.type] = (acc[shield.type] || 0) + shield.value;
      return acc;
    }, {} as Record<string, number>);
  }

  /**
   * 生成护盾ID
   */
  private generateShieldId(): string {
    return `shield_${this.owner.getId()}_${os.time()}_${math.random(
      1000,
      9999
    )}`;
  }

  /**
   * 启动护盾更新定时器
   */
  private startShieldUpdateTimer(): void {
    // 每秒更新一次护盾状态
    const timer = this.owner.getComponent(TimerComponent);
    if (timer) {
      timer.addTimer("shield_update", 1.0, true, () => {
        this.updateShields();
      });
    }
  }

  /**
   * 更新护盾状态（处理时效性）
   */
  private updateShields(): void {
    const shields = this.shieldData.getShields();
    let hasExpired = false;

    for (const shield of shields) {
      if (shield.remainingTime > 0) {
        shield.remainingTime -= 1;

        if (shield.remainingTime <= 0) {
          shield.isActive = false;
          hasExpired = true;
          this.owner.emit("shield:expired", { shield });
        }
      }
    }

    if (hasExpired) {
      // 移除过期的护盾
      const activeShields = shields.filter(
        (s) => s.isActive || s.remainingTime === -1
      );
      this.shieldData.setShields(activeShields);
    }
  }

  /**
   * 组件销毁
   */
  destroy(): void {
    // 清理定时器
    const timer = this.owner.getComponent(TimerComponent);
    if (timer) {
      timer.removeTimer("shield_update");
    }

    super.destroy();
  }
}
```

## 🔗 系统集成

### 与伤害系统集成

```typescript
/**
 * 伤害系统中的护盾处理
 */
export class DamageProcessor {
  static processDamage(
    target: Entity,
    damage: number,
    damageType: string
  ): number {
    // 获取护盾逻辑组件
    const shieldLogic = target.getComponent(ShieldComponent);

    if (shieldLogic) {
      // 护盾吸收伤害
      damage = shieldLogic.processDamage(damage, damageType);
    }

    // 返回剩余伤害给生命值系统处理
    return damage;
  }
}
```

## 📝 使用示例

```typescript
// 1. 创建带护盾系统的单位
const unit = EntityManager.createEntity("Hero");
unit.addDataManager("Inte_Shield");
unit.addComponent(ShieldComponent);

// 2. 添加护盾
const shieldLogic = unit.getComponent(ShieldComponent);
const shieldId = shieldLogic.addShield(100, ShieldType.PHYSICAL, 30); // 100点物理护盾，持续30秒

// 3. 处理伤害
const remainingDamage = shieldLogic.processDamage(80, "physical"); // 护盾吸收80点物理伤害

// 4. 查询护盾状态
const stats = shieldLogic.getShieldStats();
print(`护盾总值: ${stats.totalValue}, 激活护盾数: ${stats.active}`);

// 5. 监听护盾事件
unit.on("shield:broken", (data) => {
  print(`护盾破碎: ${data.shield.id}`);
});
```

## 🎯 设计优势

### 1. **符合现代 ECS 架构**

- 数据与逻辑完全分离
- 组件职责单一明确
- 依赖注入机制清晰

### 2. **高性能设计**

- 简化的数据结构
- 最小化的计算开销
- 事件驱动的更新机制

### 3. **易于扩展**

- 新增护盾类型只需修改枚举
- 新增业务逻辑只需扩展 LogicComponent
- 完全的类型安全支持

### 4. **易于测试**

- DataManager 可独立测试数据操作
- LogicComponent 可独立测试业务逻辑
- 组件间通过接口交互，便于 Mock
